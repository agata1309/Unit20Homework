    About The Project

    This project was a part of the UX/UI boot camp at Northwestern University in Evanston, IL.>

    Chicago Pet Rescue Website is Non-Profit organization in Chicago. It is been working since 2010.

    It provides a lot of useful information about adoption process, foster process ,pet care, lost and found etc.

    The primary goal of the redesign was to enhance the overall user experience and effectiveness of the Chicago

    Pet Rescue website. This involved several key areas to improve usability, visual appeal, and functionality

    in order to better serve potential adopters, donors, and volunteers.

    Chicago Pet Rescue Current Website

    Key Objectives of the Project:

    Simplify Processes:

    Simplify Adoption Application to make it more user-friendly and efficient.

    Create a Visually Appealing Design:

    Reflect the Organization Mission Develop a design that conveys the warmth,

    compassion, and dedication of Chicago Pet Rescue.

    Enhanced User Engagement:

    Design a layout that invites visitors to explore and engage with the site.

    This included creating engaging content sections, showcasing success stories, and highlighting featured pets.

    THE REDESIGN PROCESS

    STEP 1 : User Research

    1.Competitor Analysis/ SWOT Analysis: Three Competitors:TwoLocal and One National.

    2.Annotations : Analyze current CPR website and Identify areas for improvement.

    3.User Testing : Six Participants ,Recorded over Zoom Using current CPR website to identify pain points.

    Problem Statement :

    Chicagoans interested in adopting an animal from the Chicago Pet Rescue (CPR) website need improved

    navigation structure, search features, and more engaging visuals in order to achieve a more intuitive way to

    learn about available animals and complete the adoption process.

    STEP 2 :Definition & Synthesis:

    Brand Identity

    Cartoon is fun and friendly but not professional. Red heart & cardiogram evoke rescue but may be confused for medical.

    Propose a new, modern logo that maintains the spirit of the original

    Current Logo
    Proposed Logo

    Available Animals & Pet Profile

    Disorganized and cluttered. Difficult to get information at a glance. Good photos but difficult to navigate or find.

    Implement a grid and create hierarchy. Emphasize pet photos and make easier to view.

    Adoption Application

    Form is too low on page, confusing for users. Asks for too much information & is overwhelming.

    Simplify application form and use progressive disclosure with progress tracker.

    STEP 3 : Ideation

    Based on my gathered research data, I formulated the following Persona:

    User Persona

    Name: Allie Katz

    Age: 26

    Gender: Female

    Occupation: Biomedical Engineer

    Location: Chicago, IL

    User Persona Needs & Wants

    Animal-lover, Wants to give a home to an animal

    Feeling lonely, wants companionship

    Pain Points

    Does not want to spend too much money for a a cat from a breeder.

    Does not want to coordinate with a shelter far away, wants local.

    Does not want an elderly cat because she is afraid of high vet bills.

    STEP 5 : User Testing + Outcomes

    User Testing

    3 participants, 6 user tests

    3 tasks, 3 follow up questions

    A/B Testing

    15 participants.

    5 A/B questions

    What was found?

    All users successfully completed tasks.

    Overall Positive Feedback.

    Areas for improvement

    Adoption Information page does not need a Learn More button since you are already on that page.

    Would like a way to share an animal with friend and family.

    STEP 6 : Conclusion + Future Opportunities

    New Features

    Hero section focused on adoption with primary nav and buttons to the four action items: adopt, donate, foster, volunteer.

    Simple sitemap with multiple ways to reach each page using buttons & add search filters.

    Grid layout with less text, more images and graphics, and consistent text hierarchy.

    Uniform photo sizes have been applied for a clear, easy-to-navigate layout.

    Final Thoughts

    The new website will benefit the users and the Chicago Pet Rescue.
